# JavaScript API Dashboard

## Overview
This project is a JavaScript API Dashboard built as a single-page web application.  
It displays multiple mini apps in a responsive grid layout, with each section fetching and displaying data from a different public API.

The goal of this project is to demonstrate understanding of:
- Asynchronous JavaScript (`fetch`, `async/await`)
- Working with real-world APIs
- DOM manipulation
- Error handling and loading states
- Modular and organized JavaScript code

## How to Run
1. Clone this repository
2. Open `index.html` in your browser (or use Live Server)
3. Click any button to load data from its API

No API keys are required.
